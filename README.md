# SporTriviaSDK-android

Android library for embedding SporTrivia custom games in Android apps.

## Requirements

- Android `minSdk` 24 (Android 7.0+)
- Java 8 source compatibility
- A SporTrivia partner license (access key + secret key issued by the SporTrivia team)

## Installation (JitPack)

**Step 1.** Add JitPack to your project's `settings.gradle`:

```groovy
dependencyResolutionManagement {
    repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
    repositories {
        google()
        mavenCentral()
        maven { url 'https://jitpack.io' }
    }
}
```

**Step 2.** Add the dependency in your app `build.gradle`:

```groovy
dependencies {
    implementation 'com.github.dornanchris:SporTriviaSDK-android:1.0.0'
}
```

## Quick start

```java
// 1. Configure (typically in your Application.onCreate())
SporTriviaCredentials creds = SporTriviaCredentials.fromAssets(
    getApplicationContext(), "sportrivia.properties"
);
SporTriviaSDK.configure(
    new SporTriviaConfiguration.Builder(creds).build()
);

// 2. Launch a game
SporTriviaSDK.launchCustomGame(context, "NYI_Top5A", Sport.NHL, myDelegate);
```

See the partner setup guide in the iOS repo: <https://github.com/dornanchris/SporTriviaSDK/blob/main/PARTNER_SETUP.md> — it covers credentials, IAM scoping, and the full integration flow on both platforms.

## Local development

The repository does not check in the Gradle wrapper jar (it is regenerated by JitPack at build time). Before running `./gradlew` locally for the first time, generate the wrapper once:

```bash
gradle wrapper --gradle-version 8.10.2 --distribution-type bin
```

Opening the project in Android Studio will also sync and create the wrapper automatically.
